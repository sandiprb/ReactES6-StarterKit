#React Workspace starterkit

**Includes**
- React
- Bable ES6 Preset
- Styling with [Post Css](https://github.com/postcss/postcss)
- Linting with eslint
- wepack-dev-server

<br/>

```
git clone git@github.com:sandiprb/ReactES6-workspace.git
```
 <br/>

Install dependencies
 

    cd ReactES6-workspace 
    npm install

 <br/>
 
 Build

    npm run build

 
 <br/>
 
 Run

    npm run dev


 <br/>
 
Watch 

    npm run watch
